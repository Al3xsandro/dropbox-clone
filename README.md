# Dropbox Clone UI
Dropbox é um serviço para armazenamento e partilha de arquivos que foi base de um clone ui desenvolvido através de orientações da rocketseat

<h3>Tecnologias</h3>

<h5>Front-end</h3>
<ul>
  <li>Reactjs</li>
  <li>Nextjs</li>
  <li>Styled Components</li>
  <li>Typescript</li>
</ul>

* `https://dropboxclone-psi.vercel.app/`
